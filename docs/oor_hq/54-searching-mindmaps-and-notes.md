# Searching and Paginating Mindmaps and Notes

Inside each project, the **Mindmaps** and **Notes** tabs have search bars and pagination to help you find content quickly when a project grows large.

---

## Searching Mindmaps

1. Open a project and click the **Mindmaps** tab
2. A search bar appears in the toolbar (to the left of the Import and New buttons)
3. Type any part of a mindmap name
4. The list filters instantly — no page reload required
5. The tab badge updates to show how many results match out of the total (e.g. **3 / 12**)

Clearing the search bar shows all mindmaps again.

---

## Paginating Mindmaps

Mindmaps are shown **12 per page**. When a project has more than 12 mindmaps, pagination controls appear below the list.

- Use the **Previous** and **Next** buttons to move between pages
- The current page number is shown between the controls
- Changing the search term resets to page 1 automatically

---

## Searching Notes

1. Open a project and click the **Notes** tab
2. If the project has at least one note, a search bar appears at the top of the tab
3. Type any part of the note content or author name
4. The list filters instantly
5. The tab badge updates to show matching results out of the total

---

## Paginating Notes

Notes are shown **20 per page**. When a project has more than 20 notes, pagination controls appear below the list.

- Use the **Previous** and **Next** buttons to move between pages
- Changing the search term resets to page 1 automatically

---

## Notes

- Search on both tabs is client-side — it searches through all mindmaps/notes already loaded for the project, not just the current page
- The Kanban Boards tab has the same search and pagination behaviour — all three tabs work consistently
- Search is case-insensitive
