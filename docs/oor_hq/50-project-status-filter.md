# Filtering Projects by Status

The Projects index page shows all of your active projects by default. You can use the status filter to view projects at different stages — including archived ones.

---

## Who Can Use the Filter

Any user with access to the Projects section can use the status filter. The filter affects only what you see on screen — it does not change any project data.

---

## Available Status Options

| Status | What it shows |
|--------|--------------|
| **All Active** | Projects that are active, on hold, or completed — everything except archived. This is the default view. |
| **Completed** | Projects with status set to "Completed". |
| **On Hold** | Projects that have been paused. |
| **Archived** | Projects that have been archived. Archived projects are hidden from the default view. |

---

## How to Filter

1. Go to **Projects** from the left navigation
2. Locate the **Status** dropdown in the toolbar (to the right of the search bar)
3. Select the status you want to see
4. The list updates immediately — no page reload required

The filter combines with the search bar. For example, you can filter by **Archived** and then search by name to find a specific archived project.

---

## Returning to the Default View

Select **All Active** from the status dropdown to go back to the default view, which hides archived projects.

---

## Notes

- Archived projects do not appear in **All Active** — they must be viewed by selecting **Archived** from the filter
- The filter selection resets when you navigate away from the Projects page
- Pagination resets to page 1 whenever you change the status filter or search term
