# Restoring an Archived Project

Restoring a project moves it back to **Active** status. It reappears in the default **All Active** view and becomes fully editable again.

---

## Who Can Restore a Project

Only users with the **Admin** or **Full Access** role within your company can restore a project.

---

## What Restoring Does

- Sets the project status back to **Active**
- Removes the read-only restriction — all boards, mindmaps, and notes become editable again
- The project reappears in the default **All Active** view
- All members, boards, and content remain unchanged

---

## How to Restore a Project

1. Go to **Projects**
2. Open the **Status** filter dropdown and select **Archived**
3. Locate the project you want to restore
4. Click the three-dot menu (**⋮**) on the project card
5. Select **Restore**
6. A confirmation dialog asks you to confirm — click **Restore Project**

The project is immediately moved back to active status.

---

## Notes

- After restoring, the filter returns to showing the **Archived** view — switch back to **All Active** to see the restored project
- All content and members are exactly as they were before archiving — nothing is lost during archive or restore
- If you want to permanently remove a project instead, see [Permanently Deleting a Project](53-permanently-deleting-a-project.md)
