# Viewing Shared Projects

When someone shares a project with you, it appears in your Projects page under a dedicated tab.

---

## The "Shared with me" Tab

1. Go to **Projects** in the main navigation
2. If you have any shared projects, a **Shared with me** tab appears at the top of the page
3. Click the tab to see all projects shared with you (from any company)

The tab is hidden when you have no shared projects.

---

## What You Can See and Do

The boards, mindmaps, and notes inside a shared project are all accessible — what you can do depends on your assigned role:

| Feature | Viewer | Collaborator | Admin |
|---------|--------|-------------|-------|
| View Kanban boards | ✅ | ✅ | ✅ |
| Create/edit Kanban cards | ❌ | ✅ | ✅ |
| View Mindmaps | ✅ | ✅ | ✅ |
| Add/edit Mindmap nodes | ❌ | ✅ | ✅ |
| View Notes | ✅ | ✅ | ✅ |
| Add Notes | ❌ | ✅ | ✅ |
| Invite/remove project members | ❌ | ❌ | ✅ |

---

## Losing Access to a Shared Project

If a project admin removes you from the project or revokes your invite, the project will disappear from the **Shared with me** tab and you will no longer be able to access its contents. Any board-level memberships you held inside that project are also removed automatically.
